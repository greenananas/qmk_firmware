# Handwired Kura

This firmware is for a handwired Kura using a QMK Proton C.

Keyboard Maintainer : Kura  
Hardware Supported: [QMK Proton C](https://qmk.fm/proton-c/)

## Pinout

The following pins are used:
- Columns:
    - Column 0: A0
    - Column 1: A1
    - Column 2: A2
    - Column 3: A4
    - Column 4: A5
    - Column 5: A6
    - Column 6: A7
    - Column 7: A8
    - Column 8: B8
    - Column 9: B13
    - Column 10: B14
    - Column 11: B15
    - Column 12: B9
    - Column 13: B10
    - Column 14: B11
    - Column 15: B12

- Rows:
    - Row 0: B0
    - Row 1: B1
    - Row 2: B2
    - Row 3: B3
    - Row 4: B4
